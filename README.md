# Tesla Homepage Clone

A Tesla-inspired multi-page website built using HTML and CSS.

## Features

- Responsive navigation bar
- Tesla-inspired UI
- Full-screen vehicle carousel
- Animated marquee sections
- Vehicle information cards
- Embedded promotional video
- Energy page
- Charging page
- Shop page
- Sign In and Sign Up pages
- Hover animations and smooth transitions

## Technologies Used

- HTML5
- CSS3

## Project Structure

- Vehicles (Homepage)
- Energy
- Charging
- Shop
- Sign In
- Sign Up
